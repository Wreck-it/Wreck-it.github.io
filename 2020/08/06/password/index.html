更改教务系统密码的方法
Created 2020-08-06| Updated 2020-08-06|学校学习
Post View:

众所周知，我们学校教务系统改密码的按键就是无效按键，所以记录一下改密码的过程

使用IE浏览器

如果没有就去下载一个

由于不知道是下载版本的差别还是windows系统的差别，有两种方式


第一种：

打开IE浏览器，选择右上角的小齿轮设置选项

在设置菜单中找到兼容模式的窗口，将教务系统登录后的链接复制到

然后退出教务系统，重新进入即可修改密码


第二种：

右键点击IE浏览器的图标，选择属性

点击兼容性，选择以兼容模式运行这个程序，选择winxp3，确定即可


如果两种方式都不可以，可能是你看到这篇的时候，某些东西又变了
就搜索IE兼容模式寻找现在的方法即可
为了不让同学偷看成绩，頑張ります
