没有学习again
Created 2020-07-28| Updated 2020-07-28|Diary
Post View:

上午去医院看了看肚子那里，肾那里，之前跳绳跳个两分钟，跑步跑个400米，走路快了就会痛，所以去内科挂了号，要照b超，缴费排了好久队，那个窗口的姐姐还没把我申请单给我，真的是，这个业务能力。然后b超排队又排了半天，照的时间倒是挺短，那个照的医生也说国毛莫子问题哎，然后我就放心了。给内科的医生看，也说开点调理肠胃的药就好，太好了
中午回的太晚，就煮了面条，我煮的，妈妈说怕坨到一起，我照着在网上看的，过了凉水，果然吃完那碗都没有坨，加上木瓜条，白菜泡菜，我还切了几个奈李，还不错。
下午跟妈妈去了趟银行，对了，感觉医院拿药的人的工作挺好的，不危险也稳定不紧绷，可是妈妈说沈阳药科他们出来就是做这个，专业对口，感觉大材小用了，学那么多东西就来当个配药的，太浪费了。去银行拿给瞳瞳的钱，胎压的那个报警了，倒车也坑噔一下，然后回家路上慢慢开，又没事了，爸爸后来回来去溜了溜也说没什么事了，妈妈说可能是我昨天开车，停车的时候挤到轮胎了，下次要注意

看了七月与安生，我看到了人生抉择的问题，是稳定正统还是去闯，去闯要承担什么，或者成长要承担什么，虽然主线可能是爱情吧，这些也值得探讨
又没背单词
晚上去瞳瞳家了，逗瞳瞳在那个钻的彩色筒，围着那个转来转去，小朋友力气还挺大，抓着妈妈一起看超级飞侠哈哈哈哈哈哈哈哈哈哈哈哈哈
晚上回家又回答小唐的问题，让我深刻意识到，学习就是要自己去做，去想答案这里为什么这样，去搜去干什么，别人把步骤都一步步写清了，自己就要跟着走走，做题楽しい！

可笑的是，说着要开始学习的我，第四天了，还是什么没学，明天答应爸爸去游泳，上午必须必须学，绩点绩点啊
